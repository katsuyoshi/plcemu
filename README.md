plcemu
======
PLC comunication emulator


MITSUBISHI
======

FX (FX3U ENET)
-------
Run with the binary protocol
```
$ cd mitsubishi/fx
$ ./fxbin.rb
```

The ascii version is not implemented now.

AnS (AJ71E71)
-------
Run with the binary protocol
```
$ cd mitsubishi/ans
$ ./ansbin.rb
```

Run with the ascii protocol
```
$ cd mitsubishi/ans
$ ./ansascii.rb
```
