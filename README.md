plcemu
======
PLC comunication emulator


MITSUBISHI
======

FX (FX3U ENET)
-------
run with binary protocol
```
$ cd mitsubishi/fx
$ ./fxbin.rb
```

The ascii version is not implemented now.

AnS (AJ71E71)
-------
run with binary protocol
```
$ cd mitsubishi/ans
$ ./ansbin.rb
```

run with ascii protocol
```
$ cd mitsubishi/ans
$ ./ansascii.rb
```
